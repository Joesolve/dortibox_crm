Import Data from Excel
Instructions: Upload an Excel file with a sheet named "Service Log" to update customer data. The first row should be a header row, and data should start from row 2.
Accepted formats: .xlsx, .xls
File Preview
Expected Column Names

Your Excel file's "Service Log" sheet should have these columns:

  • Number - Customer number
  • Customer Name - Customer name
  • Address - Street address
  • Type - Domestic/Commercial
  • Bin Size - Bin size (50, 120, etc.)
  • Ward - Ward number
  • Sales Rep - Sales representative
  • Payment Type - Prepaid/Postpaid
  • Month Acquired - Month acquired
  • Active in Target Month? - Yes/No
  • Mon, Tue, Wed, Thurs, Fri, Sat - Pickup days (X = yes)
  • Subscription End - End date (optional)
Backup Data

Download a backup of all current customer data before making changes.

Tips
  • Always backup your data before importing new files.
  • Use Preview to check your file before importing.
  • Update/Add mode matches customers by their Number.
  • Replace All will delete ALL existing customers first.
  • The header row should be in row 2 (row 1 can have a title).
